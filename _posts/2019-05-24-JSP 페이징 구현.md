# 페이징 카운트 블록

## 페이지 블록의 시작번호

(현재블록 - 1) * 블록단위 + 1

1블록 => (1-1) * 10 + 1 => 1
2블록 => (2-1) * 10 + 1 => 11
3블록 => (3-1) * 10 + 1 => 21

## 페이지 블록의 끝번호

블록시작번호 + (페이지블록단위 - 1)

1블록 => 1 + (10 - 1) => 10
2블록 => 11 + (10 - 1) => 20
6블록 => 51 + (10 - 1) => 61
