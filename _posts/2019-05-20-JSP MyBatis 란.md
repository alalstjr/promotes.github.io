# MyBatis

1. 개발자가 지정한 SQL, 저장프로시저를 지원하는 프레임워크

2. 프로그램 소스 안에 SQL문을 작성하던 기존 JDBC 방식과 달리 SQL문을 프로그램에서 분리하여 XML 파일에 별도로 작성

3. MyBatis의 장점
  - 코딩량 절감
  - 간편한 유지보수 (SQL을 변경하고자 할 경우 기존처럼 프로그램을 수정하는 것이 아니라 
  XML 파일의 SQL문만을 변경하면 되기 때문에 SQL변환이 자유로움)
