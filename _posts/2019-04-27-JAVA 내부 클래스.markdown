# 내부 클래스

<p>
  클래스 안에 선언된 클래스 <br/>
  특정 클래스 내에서만 주로 사용되는 클래스를 내부 클래스로 선언한다. <br/>
  <b>GUI애플리케이션(AWT, Swing)의 이벤트처리에 주로 사용됨</b> <br/>
  내부클래스도 클래스이기 떄문에 <b>(외부클래스$내부클래스.class)</b> 로 컴파일 됨
</p>

## 장점

<p>
  내부 클래스에서 외부 클래스의 멤버들을 쉽게 접근할 수 있다. <br/>
  코드의 복잡성을 줄일 수 있다.
</p>
