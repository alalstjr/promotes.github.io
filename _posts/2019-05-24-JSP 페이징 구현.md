우선 MySQL 프로시저를 활용하여 빈 테이블에 Data 900개를 체워 놓습니다. <br/>
10개씩 한페이지당 끈어서 보여주기 위해서 Test용 Data 입니다.

데이터를 초기값부터 10개를 가져오는 SQL
> SELECT * FROM {table} LIMIT 10;

데이터를 지정된 위치부터 10개를 가져오는 SQL
> SELECT * FROM {table} LIMIT 10, 10;
